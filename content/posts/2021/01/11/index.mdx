## Gatsby.jsにTailwindCSSを使ってリストを作る方法

- 前提
- 簡単に出来るリストの作り方

### 前提

- `gatsby-theme-blog` を利用している
- `TailwindCSS` を利用している
- `MDXProvider` を使って装飾している

### 簡単に出来るリストの作り方

- `MDXProvider` を使っている。
- リストのインデントの表現は、[こちら](https://tailwindcss.com/docs/padding#class-reference)
- 実際に作ってみたものが以下になります。

```jsx
import React from "react"

const Layout = {
    ul: props => <ul className='list-disc list-inside' {...props} />,
    li: props => <li className='pl-4' {...props} />
}

export default Layout;
```

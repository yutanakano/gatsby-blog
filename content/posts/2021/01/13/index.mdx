Gatsby.jsでfaviconを適用させる方法についてメモを残しておきます。

## Gatsby.jsでfaviconを設定する方法
- pluginのインストール
- gatsby-configに設定を追加
- 画像の配置

### pluginのインストール

- `gatsby-plugin-manifest` を インストールする

```
npm install --save gatsby-plugin-manifest
```

### gatsby-configに設定を追加

- `plugins` の中に以下を追加する(faviconを配置するディレクトリのパスを明示的に記載する)

```
plugins: [
+    {
+      resolve: `gatsby-plugin-manifest`,
+      options: {
+        icon: `content/assets/faviconまでのpath`,
+      },
+    },
]
```

### 画像の配置

今回は `content/assets/` 配下に設置するようにしました

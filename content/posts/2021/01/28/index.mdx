## Gatsby.jsにカテゴリ機能を実装する中でハマった話

`frontmatter` からデータを取得する場合は事前に `frontmatter` にkey(ここでいう `category` と `tags` です)を登録しておく必要があります。

以下の `+` が登録したkeyです

```
---
title: example
date: 0000-00-00
image: ./image.png
imageAlt: example
+ category: example
+ tags: ['example-tag', 'example-post', 'example-category']
---
```

一度追加したら認識させるために `yarn run dev` を再実行する必要があります。

上記の対応を行うことでQueryを実行すると `category` と `tags` の値を取得できるようになります。
